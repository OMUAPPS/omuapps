# OMUAPPSの導入方法

現在は**Windowsのみ対応**しています

このページでは、OMUAPPSをダウンロードしてインストールするまでの手順を説明します
初期設定やアプリの使い方については、別ページ（[初期設定ガイド](%DOCS_ROOT%/guide/setup)）をご覧ください

## 1. ダウンロード手順

[ダウンロードページ](/download)からインストーラーを入手します

ブラウザによっては、初回ダウンロード時に警告が表示される場合があります
特に Microsoft Edge では以下のようなメッセージが出ることがあります

![Edgeの警告画面](/img/download_warning_edge_0.png)

> [警告が出て止まった場合はこちら](%DOCS_ROOT%/guide/download-warning)

## なぜ警告が出るのか

この警告は、まだ広くダウンロードされていないことを理由に表示される場合があります

たとえば、遠隔操作を行うアプリがあった場合でも、ウイルス検出システムはそれが「ユーザーが意図して使う正規の機能」なのか、「悪意ある目的で利用されている」のかを判断できません

同様に、遠隔操作のような機能がなくても、セキュリティチェックの基準が厳しくなっているため、WindowsのSmartScreenや各ブラウザは、ダウンロード実績の少ないアプリを警告対象とします

OMUAPPSは、アプリを安心して利用できるよう、個人開発でも[安全にアプリを配布・導入できる仕組み](%DOCS_ROOT%/guide/security)を提供しています

## 2. インストール手順

ダウンロードしたインストーラーを起動すると、次のような画面が表示されます
基本的には「次へ」をクリックしていくだけで完了します

特別な理由がない限り、入力欄はそのままで変更しないことをおすすめします

## 3. セットアップ

起動後はこのような画面になります

![ダッシュボード](/img/dashboard_install.png)

基本的各手順には説明がありますので読みながら進めていけばセットアップが完了します

## 4. アプリの入れ方

気になったアプリを導入してみましょう

アプリはダッシュボード内の「アプリを探す」タブから導入することができます

アプリの紹介や詳しい使い方は[アプリ一覧](%DOCS_ROOT%/app/)をご覧ください
