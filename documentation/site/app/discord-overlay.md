# Discordオーバーレイ

ちょっと変わったDiscordオーバーレイです

![Discord Overlay](/img/discord-overlay.png)

## 安定して表示できるDiscordオーバーレイ

稀にある「表示されない」問題が、このアプリでは少し仕組みが違うため、表示できないことが少なくなっています

## PNGTuber+アバターに対応

このアプリでは通常のアイコンだけでなく、あなたの持っているPNGTuber+アバターも使うことができるので、今まで不可能だったPNGTuber+でのコラボ配信ができます

<iframe src="https://store.steampowered.com/widget/2596880/" frameborder="0" width="100%" height="190"></iframe>

## 使うには

配信アプリ管理ツール「[OMUAPPS](/)」内の「アプリを探す」タブから導入ができます

面倒なこともなく設定なしで簡単に追加することができます
