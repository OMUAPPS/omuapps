# アプリ一覧

## Discordオーバーレイ

- 安定して起動できるDiscordオーバーレイ
- DiscordのボイスチャットをOBSに表示することができます
- PNGTuber+アバターに対応。あなたの持っているPNGTuber+アバターでコラボ配信ができます

## リアクション

- Youtubeのリアクションボタンで出る効果を好きなように出すことができます
- 画像を置き換えることができます

## リプレイ

- 過去にした配信やYoutubeの動画を流すことができます
- OBS側ではモザイクをかけたり、時間などの情報を表示できるので同時視聴にぴったりです
- 音をOBSとアプリの別々で管理することができます

## ルーレット

- 手動で追加するほか、特定のキーワードを設定して募集してコメントでエントリーして抽選することもできます
