## API一覧

OMUAPPSのアプリはこれらの機能を使うことで作られています

### [レジストリ](%DOCS_ROOT%/api/registry)

データをアプリ間で同期・保持する事ができます

### [シグナル](%DOCS_ROOT%/api/signal)

アプリ間で任意のデータで通知を出し合う事ができます

### [テーブル](%DOCS_ROOT%/api/table)

キーと値を持った大量のデータを保持することができます

### [アセット](%DOCS_ROOT%/api/asset)

大きいデータをファイルとして保持する事ができます

### [ダッシュボード](%DOCS_ROOT%/api/dashboard)

ファイルのドラッグ&ドロップや、音声認識を要求する事ができます

### [HTTP](%DOCS_ROOT%/api/http)

ブラウザの制限を超えてHTTPリクエストを送信したりWebSocketを開く事ができます

### [セッション](%DOCS_ROOT%/api/session)

子アプリのためのトークンを生成したり、他のアプリの接続を待機する事ができます
