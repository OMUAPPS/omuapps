# 権限とは？

アプリが**何をするか**を導入する前に確認するための機能です。
このときに導入したくないと感じればすぐ**キャンセルすることが可能**です。

## もし権限がなかったら？

本来アプリはインストール時点で必要でもないすべての機能にアクセスすることができます。

そこで、権限の仕組みがあることによって、本来見えない何を必要としているかを事前に一覧で確認できるため、信用できるかを自分で判断しやすくなります。

詳しくはセキュリティについての記事をご参照ください

[OMUAPPSのセキュリティ](https://omuapps.com/docs/guide/security/)
